# Gramathu Boutique - Premium Saree E-commerce

Traditional Elegance with Modern Style. A luxurious fashion boutique experience featuring smooth animations, premium typography, responsive layouts, and an integrated admin control panel.

## Key Features
- **Premium UX/UI**: Glassmorphism effects, auto-sliding luxury banner, smooth page transitions, skeleton loaders.
- **Dynamic Cart & Wishlist**: Badge count indicators, instant item updates, custom toast notifications.
- **Erode Shipping Logic**: Free shipping for Erode district, configurable shipping fees for other locations.
- **Interactive Auth**: Password strength validator (Weak, Medium, Strong) with real-time requirements checking.
- **Full Admin Panel**: Managing Sarees, Categories, Orders, Customers, Inventory, Banners, and System Settings.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm (v9+)

### Installation
1. Install all dependencies across the workspace:
   ```bash
   npm run install-all
   ```

2. Start the development servers (Client on `http://localhost:5173` and Server on `import.meta.env.VITE_API_URL`):
   ```bash
   npm run dev
   ```
